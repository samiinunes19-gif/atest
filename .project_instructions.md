# Atestado Médico 24 Horas

## Visão Geral
Site de teleconsulta médica online para emissão de atestados médicos. Funil de vendas com 7 etapas, pagamento PIX via MasterPag, validação de comprovante por IA.

## Stack Técnica
- Next.js 14.2.28 (App Router), React 18.2, TypeScript 5.2
- Tailwind CSS 3.3.3 + shadcn/ui (Radix UI)
- Prisma 6.7.0 + PostgreSQL
- Framer Motion para animações
- Lucide React para ícones

## Design System
- **Cor primária:** #125F39 (emerald escuro, HSL 152 68% 22%)
- **Fonte:** Plus Jakarta Sans (Google Fonts, pesos 200-800)
- **Tema:** Light only (sem dark mode)
- **Border radius:** 0.75rem (12px) padrão, botões rounded-full
- **CSS Variables:** HSL-based via :root

## Estrutura de Páginas
- `/` - Homepage (hero carousel, how-it-works, features, testimonials, FAQ, CTA)
- `/requisicao` - Funil 7 etapas (2231 linhas, client-side)
- `/verificar` - Consulta validade atestado
- `/empresa` - Sobre nós
- `/garantia` - Política de reembolso
- `/politica-privacidade` - LGPD
- `/termos-de-uso` - Termos

## APIs
- `/api/payment/pix` - Gera QR Code PIX via MasterPag
- `/api/validate-comprovante` - Valida comprovante via AbacusAI (gpt-4.1-mini)

## Database
- Model PixCache: cache de PIX por CPF, expira em 30min
